The class NumC is doing the convertion of the finantional flow data to the defined format for the comfortable work of the finantional application. I has static methods that can be used:

  • toData - gets an input, removes spaces from the string, replaces all commas to be dots and returns the decimal number (floating point number)
  • toBuh - check up the input to be matching with requirements ? returns the string with number and optional currency : returns input without any changes
  • convertToBuh - localize input in accordance with locale and returns the result string in the format 'number plus currency' (currency is optional)
  • isNumeric - check whether an input is a number, returns boolean value
  • toSymbol - returns strings with currency symbol by the input with the currency abbr
  • toCurrency - returns strings with currency abbr by the input with the currency symbol
  • calcSummFromRows - by the class of the element in the DOM gets data and returns the sum of elements of the data array
  • getFALDates - gets the first and last element's date and returns an object with these data
